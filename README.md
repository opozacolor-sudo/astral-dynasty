# Astral Dynasty - Ștergere cont

Pentru a șterge contul și toate datele asociate, urmează acești pași:

1. Deschide aplicația **Astral Dynasty**
2. Mergi la **Profil** (iconița din dreapta jos)
3. Derulează jos până la butonul **Șterge contul**
4. Confirmă acțiunea

Contul și toate datele tale vor fi șterse imediat și ireversibil.
